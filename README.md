# vertical-align package

`ctrl-cmd-/` to align operators.

![vertical-align](https://raw.github.com/adrianlee44/atom-vertical-align/master/demo.gif)

## Examples
From:
```coffeescript
foo = "bar"
test = "notest"
hello = "world"
```

To:
```coffeescript
foo   = "bar"
test  = "notest"
hello = "world"
```

From:
```coffeescript
random =
  troll: "internet"
  foo: "bar"
  bar: "beer"
```

To:
```coffeescript
random =
  troll: "internet"
  foo:   "bar"
  bar:   "beer"
```
